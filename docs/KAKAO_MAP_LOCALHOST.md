# 로컬(localhost)에서 카카오맵이 안 나올 때

JavaScript 키를 넣었는데도 "지도 로딩 중..." 또는 "지도 로드 시간 초과"가 나오면 **도메인 등록**과 **카카오맵 사용 설정**을 확인하세요.

## 1. JavaScript SDK 도메인 등록

1. [developers.kakao.com](https://developers.kakao.com) 로그인
2. **내 애플리케이션** → 사용하는 앱 선택
3. **앱 설정** → **플랫폼** → **Web** (없으면 Web 추가)
4. **JavaScript SDK 도메인**에 **정확히** 추가:
   - `http://localhost:3000`
   - (다른 포트면 예: `http://localhost:3001`)
5. **저장**

## 2. 카카오맵 사용 설정 켜기 (필수)

도메인만 추가해서는 부족할 수 있습니다. **카카오맵 API 사용 설정**이 꺼져 있으면 지도가 로드되지 않습니다.

1. 같은 앱에서 **제품 설정** (또는 **앱** 메뉴)
2. **카카오맵** 선택
3. **사용 설정**을 **ON**으로 변경 후 저장

(콘솔 UI가 다르면: **앱** → **카카오맵** / **지도** 메뉴에서 “사용 설정” 또는 “상태 ON” 확인)

## 3. .env.local 확인

```env
NEXT_PUBLIC_KAKAO_MAP_KEY=여기에_JavaScript_키
```

- **앱 키** 탭의 **JavaScript 키**를 복사해서 넣으면 됩니다.
- 수정 후 **개발 서버를 한 번 재시작**하세요.

## 4. 그래도 안 되면

- 브라우저 개발자 도구(F12) → **Console** 탭에 빨간 에러가 있는지 확인
- **Network** 탭에서 `dapi.kakao.com` 요청이 **성공(200)** 인지 확인
